# Control UI Rebuild Reference

The full control-surface rebuild uses `UI-redo-sample.png` in the repository root
as the visual reference. The target direction is:

- top command/status bar with LiveLayer identity, local route state, OBS state,
  preview output, setup, and operator avatar
- fixed left icon rail for major operator sections
- template/saved-work column on the left
- large central preview monitor with background/safe-area controls
- editor tabs below the monitor
- right-side on-air actions, layout, and auto-hide inspector

The rebuild should preserve `/output`, Take/Clear behavior, local-first storage,
Saved Graphics, People, Rundowns, import/export packs, assets, and scripture
lookup while making the full web control interface match the production-console
feel of the reference.
